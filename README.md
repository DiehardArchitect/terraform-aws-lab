# AWS Lab Infrastructure with Terraform

Automated AWS infrastructure for my college lab environment using Terraform.

## What I Built
- Ubuntu 24.04 instance for Database Systems class
- Automated deployment with Terraform
- AWS CLI integration

## Tech Stack
- Terraform
- AWS EC2
- AWS CLI

## Usage
```bash
terraform init
terraform plan
terraform apply
```
